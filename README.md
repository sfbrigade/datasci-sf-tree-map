# sf-voters
Mapping San Francisco

Make Choropleths by:
- census tracts
- city assessor lots (blocklots)
- election precincts
- neighborhoods - planning dept
- neighborhoods - sffind
- neighborhoods - health dept
- supervisor districts
- zip codes

Data from [DataSF](https://data.sfgov.org/)
